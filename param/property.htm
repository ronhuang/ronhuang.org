原生動物門：原生動物因為祇有一個細胞，所以屬於最簡單最原始的動物。這一類的動物約有100000種以上，包括鞭毛蟲、變形蟲、胞子蟲和纖毛蟲等。由於單細胞動物的細胞體非常微小，因此到十六世紀末年，顯微鏡發明後，才被人類發現。這種簡單的動物除了極少數像鞭毛蟲、眼蟲能夠自己行光合作用，獲得營養外，大多數是寄生性、腐生性或自由取食的。原生動物既然沒有嘴和消化器官，就祇能依賴細胞膜，使食物直接滲透而吸收進去。又因為沒有腳沒有手，所以頂多就是像變形蟲一樣連滾帶爬的運動，或像鞭毛蟲一樣用簡單的鞭毛在水中划動游泳了。單細胞動物對高等動物來說，多半是有害的。例如寄生人體會引起赤痢、瘧疾、昏睡病等；也能寄生在蠶的體內，引起微粒子病；寄生在蜜蜂體內引起下痢。原生動物包含鞭毛綱、原肉綱、胞子蟲綱、纖毛蟲綱、吸管綱：
鞭毛綱 ( Clas Mastigophora )：終生或僅幼期具鞭毛一枚或數枚，有自由生活及行寄生性者，生殖採縱製法。本綱所佔種類最多，如眼蟲，睡眠原蟲，毛滴蟲等。眼蟲是生物教學及實驗課最常用材料，其兼具動植物特徵，有以其為動物始祖之假說。睡眠原蟲藉昆蟲叮咬傳播，可引起非洲睡眠症。毛滴蟲是一項世界性煩擾，其可造成生殖道之感染損傷。
肉足綱( Class Sarcodina ) ：又稱阿米巴原蟲，成體皆具偽足，用於移動、捕獲獵物等，除了少數寄生性種類，其食物包括有細菌、藻類、矽藻，及其他原生動物。主要成員有變形蟲類，有孔蟲類，太陽蟲類及放射蟲類。變形蟲是生物實驗常用材料，可隨時改變體形。有孔蟲與放射蟲多海產，體外被硬殼，死後不易腐化，是海洋積砂中主要成分，也是地質探勘石油層主要依據之一。太陽蟲之體呈球形，有堅硬之放射狀偽足，大多淡水產。
孢子蟲綱 ( Class Sporozoa )：這是一群歧異性很高之寄生蟲，在其生活史中具有孢子蟲，其藉昆蟲在寄主間傳遞，最有名之例為瘧疾原蟲，約有50種，可使人類生病者共4種。
纖毛蟲綱( Class Ciliata )：在單細胞生物中，本綱展示了最高之複雜性，其軀體外表具有千百根纖毛，口區沒有纖 毛或不明顯，在游泳時可同步移動，有時體表尚可放射出許多防禦用之有毒纖毛，已如約有8000種纖毛蟲，常見者有草履蟲、喇叭蟲、吊鐘蟲等。

　

原生動物門──纖毛蟲綱：纖毛蟲綱在原生動物中體制分化最為複雜的。體面被有比較堅實之薄膜，所以有一定的體形。薄膜如在高倍顯微鏡下觀察，密佈無數杯狀之凹窪，每一凹窪各具有一六角形之邊緣。其終生具有之纖毛，基部在薄膜下，下端各連一小基粒，向上通過凹窪底部，然後突出體面。小基粒各有橫走與縱走之原纖維連絡如網。此小基粒與原纖維互相聯繫組成纖毛蟲之神經運動系統。大小二型之核亦為本綱重要特徵之一。大核主營養，小核主生殖。有絲分裂時，二核同樣分裂，但在有性生殖時，大核破裂而終歸消失，小核經幾次分裂，重新分生新的大核與小核。草履蟲便是屬於此類。

　

草履蟲沒有腦（這點我早就知道了），可是卻能遠離不喜歡的環境，而聚集在喜歡的地方。例如：
對空氣、重力等的反應：草履蟲不喜歡待在壓力高的地方，喜歡空氣流通的地方，因此不會聚集在此處。
碰到物體的反應：喜歡在物體的周圍，因此會聚集在棉花附近 。
對電的反應：利用電極，就可以看出他喜歡移向陰極 。
對熱的反應：性喜溫度適中的地方 。

　

草履蟲採用無性生殖來繁殖後代。無性生殖有很多種，例如：出芽生殖、出芽生殖、斷裂生殖、孢子生殖、 營養器官繁殖、孤雌生殖、組織培養、分裂生殖和最近流行的”複製羊”繁殖等等，而草履蟲是屬於分裂生殖。
