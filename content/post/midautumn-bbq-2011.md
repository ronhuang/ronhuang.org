睽違三年的中秋烤肉又要回來了，照慣例要邀請小寶的同事來烤肉。跟往年一樣，
歡迎攜帶女伴一起參加唷。男伴就不用勉強了。

![中秋烤肉的標誌][1]

今年仍然有一個新的網站供各位推薦中秋烤肉的物品：[中秋烤肉]。
今年仍是架在 Google App Engine 上，後端是用 Python 寫的，前端是 HTML +
JavaScript + CSS 。不過今年的網站與 [Facebook] 整合較多，而且多了成就系統。
歡迎各位上去玩玩，蒐集各式各樣的成就。
